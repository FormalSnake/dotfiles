Herdr 从 `config.toml` 读取的所有规范键都在这里平铺列出，并支持筛选。
配置的分步指南与各选项的背景说明见[配置](/zh-cn/docs/configuration/)。

随时可以打印带注释的完整默认配置:

```bash
herdr --default-config
```

自定义命令绑定（`[[keys.command]]`）是用户自定义的表，因此不在此处逐键列出，
见[自定义命令按键绑定](/zh-cn/docs/configuration/#自定义命令按键绑定)。

各键的描述目前为英文。
